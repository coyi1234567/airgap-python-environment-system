## 🚀 使用说明

### 快速部署
```powershell
# 1. 复制整个offline-repo文件夹到内网机器
# 2. 运行一键部署脚本
.\scripts\setup.ps1

# 3. 等待安装完成
# 4. 验证环境
.\scripts\verify-deployment.ps1
```

### 环境管理
```powershell
# 激活环境
conda activate base-scientific

# 安装包
conda install numpy pandas

# 导出环境
.\scripts\env-manager.ps1 -Action export -EnvName myenv
```
